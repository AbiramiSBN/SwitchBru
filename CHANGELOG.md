# Changelog

All notable changes to the SwitchBru Launcher project are documented here.

---

## [1.0.0] – Initial Public Version

### Added
- Home launcher UI with large rounded tiles
- Games, Apps, Web, and Settings sections
- Search bar with URL detection and Google fallback
- Controller and keyboard navigation
- Dynamic loading from `catalog.json`

### Games
- Support for local and external games
- HTML5 game portals (ArmorGames, Kongregate, Poki, etc.)
- Pin, search, and sort functionality

### Apps
- App launcher with dynamic JSON loading
- Added PerminiGPT AI app
- Search support

### Web
- Curated list of Switch-compatible websites
- Pinning and sorting
- Dedicated Web page UI

### Settings
- Wallpaper presets (1–10)
- Custom wallpaper URL support
- Live preview and image testing
- Text color customization
- Tile size adjustment
- Title glow toggle
- Version text customization
- Settings saved via localStorage

### UI / UX
- Glowy title bloom
- Controller-friendly spacing and sizing
- Emoji-safe icons
- Responsive layout for Switch screen

---

## [1.1.0] – Homepage Enhancements
